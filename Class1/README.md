# Flask Simple Web App

A basic Flask application with multiple routes demonstrating routing and dynamic URLs.

## Routes
- `/` - Home page
- `/contact` - Contact page
- `/about` - About page
- `/profile` - Profile page
- `/profile/<name>` - Dynamic profile page with custom name

## Installation
```bash
pip install flask
```

## Run
```bash
python app.py
```

App runs on `http://127.0.0.1:5000/` in debug mode.
